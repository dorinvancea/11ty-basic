Basic 11ty set up:

- Install: `npm install`

### Run
- `npm run dev` to start the server
- Server at [http://localhost:8080/](http://localhost:8080/)

### Build
- `npm run build` to build
- Output folder: Change the  name in the `.eleventy.js` file
